# CONTENT PERFORMANCE ANALYSIS WORKFLOW

Versão: 1.0

Status: Documento Oficial

Classificação: Workflow Corporativo

---

# OBJETIVO

Analisar continuamente o desempenho dos conteúdos publicados, gerar insights acionáveis e retroalimentar o planejamento editorial com base em evidências.

Este Workflow fecha o ciclo operacional de conteúdo do Marcos.

---

# RESULTADO ESPERADO

Ao final da execução deverão existir:

- Relatório consolidado de performance.
- Insights priorizados.
- Conteúdos vencedores.
- Conteúdos de baixa performance.
- Recomendações para o próximo ciclo.
- Base histórica atualizada.

---

# GATILHOS

- Publicação concluída.
- Encerramento de período (diário, semanal ou mensal).
- Solicitação dos fundadores.

---

# ENTRADAS

- Métricas das plataformas.
- Calendário editorial.
- Objetivos dos conteúdos.
- KPIs esperados.
- Histórico de publicações.

---

# CAPABILITIES

- Analytics & Insights
- Creative Performance
- Growth Experimentation
- Content Strategy

---

# OPERATORS

- Instagram Operator
- Facebook Operator
- PostgreSQL Operator
- LLM Operator

---

# PROCESSO

## Etapa 1

Coletar métricas das plataformas.

## Etapa 2

Consolidar indicadores.

## Etapa 3

Comparar desempenho com objetivos definidos.

## Etapa 4

Identificar padrões positivos e negativos.

## Etapa 5

Gerar hipóteses e recomendações.

## Etapa 6

Atualizar base histórica.

## Etapa 7

Encaminhar aprendizados para o Workflow de Planejamento.

---

# MÉTRICAS AVALIADAS

Quando aplicável:

- Alcance.
- Impressões.
- Engajamento.
- Compartilhamentos.
- Salvamentos.
- CTR.
- Conversão.
- Crescimento de seguidores.
- Tempo de visualização.
- Custo por resultado.

---

# SAÍDAS

- Dashboard consolidado.
- Relatório executivo.
- Lista de recomendações.
- Conteúdos referência.
- Aprendizados registrados.

---

# CRITÉRIOS DE QUALIDADE

- Dados confiáveis.
- Recomendações objetivas.
- Evidências rastreáveis.
- Comparabilidade histórica.

---

# KPIs

- Tempo de geração do relatório.
- Quantidade de insights acionáveis.
- Taxa de implementação das recomendações.
- Evolução dos indicadores.

---

# PRINCÍPIO FINAL

Todo conteúdo publicado deve produzir conhecimento para o próximo ciclo.

O objetivo deste Workflow não é apenas medir resultados, mas transformar desempenho em melhoria contínua.
